# Reddit Marketing Monitor

Personal tool to scan Reddit for posts relevant to your indie products, draft AI replies with Gemini, and post them manually.

**Products:** Kobo Note Up · txtconv

---

## Quick Start (Mac)

### 1. Install Node.js
Download LTS from https://nodejs.org

### 2. Get the project
Download and unzip, or clone this repo.

### 3. Run setup
```bash
cd reddit-marketing-monitor
bash setup.sh
```

This will:
- Create `.env.local` with auto-generated secrets
- Ask you to fill in `GEMINI_API_KEY` and `APP_PASSWORD`
- Install dependencies, migrate DB, seed products
- Start the app at http://localhost:3000

### 4. Fill in `.env.local`

```env
GEMINI_API_KEY=AIzaSy...        # From https://aistudio.google.com/apikey
APP_PASSWORD=your_password       # Any password you choose
ENCRYPTION_KEY=<auto-generated>
CRON_SECRET=<auto-generated>
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### 5. Run again
```bash
bash setup.sh
```

Open http://localhost:3000, login, hit **Scan Now**.

## Features

- 🔍 Scans Reddit for relevant posts (public API, no OAuth needed for reading)
- 🤖 AI relevance scoring with Gemini 2.0 Flash
- ✍️ AI-drafted replies with UTM tracking
- ✅ Manual approval always — nothing posts without your confirmation
- 📊 Analytics: top subreddits, keywords, reply rate
- 🔔 Email + Telegram notifications for high-relevance posts

## To also POST replies via Reddit

Reddit requires API approval to post comments. Apply at:
https://www.reddit.com/prefs/apps

Once approved, add to `.env.local`:
```env
REDDIT_CLIENT_ID=your_client_id
REDDIT_CLIENT_SECRET=your_secret
REDDIT_REDIRECT_URI=http://localhost:3000/api/auth/reddit/callback
```

Then go to **Settings → Reddit** in the app and connect your account.
